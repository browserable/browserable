---
title: "Health Check"
openapi: "GET /health"
description: "Check if the API server is running"
---

A simple endpoint to check if the API server is up and running. This endpoint does not require authentication.

<RequestExample>
```bash
curl --request GET \
  --url 'http://localhost:2003/api/v1/health'
```
</RequestExample>

<ResponseExample>
```json
{
  "success": true,
  "data": "ok"
}
```
</ResponseExample>
