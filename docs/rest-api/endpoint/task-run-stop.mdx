---
title: "Stop Task Run"
openapi: "PUT /task/{taskId}/run/stop"
description: "Stop a running task execution"
---

<RequestExample>
```bash
curl --request PUT \
  --url 'http://localhost:2003/api/v1/task/task_123/run/stop?runId=run_456' \
  --header 'x-api-key: <your-api-key>'
```
</RequestExample>

<ResponseExample>
```json
{
  "success": true
}
```
</ResponseExample>
