---
title: "List Task Runs"
openapi: "GET /task/{taskId}/runs"
description: "Get a list of all runs for a specific task"
---

<RequestExample>
```bash
curl --request GET \
  --url 'http://localhost:2003/api/v1/task/task_123/runs?page=1&limit=30' \
  --header 'x-api-key: <your-api-key>'
```
</RequestExample>

<ResponseExample>
```json
{
  "success": true,
  "data": [
    {
      "id": "run_456",
      "created_at": "2024-03-20T15:30:00Z"
    }
  ],
  "total": 1,
  "page": 1,
  "limit": 30
}
```
</ResponseExample>
